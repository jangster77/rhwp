- `line_cs_offset` vs `effective_col_x`: HWP3 wrap_precomputed용 x오프셋과 HWPX/HWP5 Square wrap용 x오프셋이 공존. cherry-pick 충돌 해결 시 `if line_cs_offset > 0.0 { col_area.x + line_cs_offset } else { effective_col_x }` 로 결합.
- HWP3 전용 로직은 `src/parser/hwp3/` 안에서만 구현 (CLAUDE.md 규칙 준수). 렌더러는 `wrap_precomputed` 플래그로만 분기.
